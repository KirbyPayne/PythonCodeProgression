Firewall Detection Engine

I defined event data through using lists and tuples, built rule functions, a rule engine, a decision engine with priority resolution, a process engine using counters while using a for loop to loop through events and make decisions using if/elif/else statements, and printed a summary that listed events,  devices and decisions.


SOC Threat Analyzer


I created a SOC Threat Analyzer using the format of procedural and modular programming with the uses of multiple functions and a main function call. Event data was stored in a list, A function was created to calculate threat scores by using a counter, a for loop to process events, if/elif/else statements were used to score threats by events and a return to send a value back to the function. A seperate function was used to calculate severity levels using if/elif/else statement linked to risk score. A third function was used to determine an action to take using if/elif/else statements to specify an action to take based on risk score. The main function was called while threats were calculated by risk score. A summary of final risk score, severity and action to take were printed.


IOC Analyzer


I created an IOC Analyzer using three datasets. I grouped the first two datasets into a list of observed and malicious IPs. A third dataset was created to show observable severity levels which included using a dictionary to store malicious IPs and severity levels. I used a counter to count malicious IPs.I created logic for an engine by using a for loop and checking ips using if/else statements. I output a summary that checked IPs, determined maliciousness, severity levels and counted malicious IPs.
